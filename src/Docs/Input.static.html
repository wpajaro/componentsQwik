Input

Componente de entrada de datos que admite múltiples tipos (texto, email, contraseña, etc.) y también puede funcionar como select dropdown.

Importación

import Input from '~/components/atoms/input';

Props

Prop Tipo Descripción Por defecto
id string Identificador único para el input —
label string Etiqueta descriptiva para el campo de entrada —
type "text" | "email" | "password" | "number" | "date" | "url" | "select" Tipo de entrada o selector "text"
placeholder string Texto de marcador de posición —
options string[] Opciones disponibles cuando type="select" —
value string Valor inicial del campo —
onChange$ QRL<(value: string) => void> Función que se ejecuta cuando cambia el valor —
disabled boolean Deshabilita el campo de entrada false

Ejemplos

Ejemplo básico de texto

<Input 
  label="Nombre completo" 
  placeholder="Ingrese su nombre" 
  onChange$={(value) => console.log(value)} 
/>

Campo de contraseña

<Input
  type="password"
  label="Contraseña"
  placeholder="Ingrese su contraseña"
/>

Selector (dropdown)

<Input
  type="select"
  label="País"
  options={['México', 'Colombia', 'Argentina', 'España']}
  onChange$={(value) => console.log('País seleccionado:', value)}
/>

Campo deshabilitado

<Input
  label="Email"
  type="email"
  value="usuario@ejemplo.com"
  disabled={true}
/>
Notas:
  • El componente automáticamente se adapta para mostrar un campo de entrada normal o un selector basado en la propiedad type.
  • Los estilos se aplican desde input.module.css y utilizan clases de Bootstrap para el formato básico.
  • Cuando se usa como selector (type="select"), la primera opción siempre será "Seleccione una opción".
  • El componente está optimizado para trabajar con Qwik y su sistema de renderizado SSR.
