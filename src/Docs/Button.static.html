Button

Componente de botón personalizable que admite múltiples estilos de color, variantes y tamaños.

Importación

import Button from '~/components/atoms/button';

Props

Prop Tipo Descripción Por defecto
label string Texto que se muestra dentro del botón “Primary Button” (u otro según color)
onClick$ QRL<() => void> Función que se ejecuta al hacer clic —
color "primary" | "secondary" | "success" | "info" | "warning" | "danger" | "light" Color del botón "primary"
type "fill" | "outline" Estilo del botón "fill"
variant "default" | "large" | "small" | "extra-small" | "bold-border" Variación de tamaño y borde "default"
disabled boolean Deshabilita el botón false

Ejemplos

Ejemplo básico

<Button label="Guardar" onClick$={() => console.log('Guardado')} />

Con color y variante

<Button
  label="Eliminar"
  color="danger"
  type="outline"
  variant="bold-border"
  onClick$={() => alert('Elemento eliminado')}
/>

Botón deshabilitado

<Button
  label="Deshabilitado"
  disabled={true}
/>
Notas:
  • Si label no se especifica, se muestra automáticamente basado en el color.
  • Los estilos se aplican desde button.module.css.
  • Se integra fácilmente con Qwik SSR desde Bun.
